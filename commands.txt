0. 从 C 盘 转入 D 盘 ，需要 输入 “D:”, 不用输入 cd D: , 从 D 盘 转入 C 盘 亦然。
1. cd --进入到某个目录下。
2. cd..  --退出当前的目录，即返回上一级的目录。
3. rd --删除，准确地说是移除(remove)，移除选定目录。
4. dir --显示当前目录下所包含的文件和目录。 -- directory : (计算机文件)目录。
5. md -- make directory -- 创建一个目录(文件夹)。
6. del -- delete -- 删除文件。      
   注意:它不能删除目录，但是当其作用于目录时，起的作用是删除该目录中的文件。
   del *.txt(或者其他类型的文件) -- 删除某一类型的文件。
